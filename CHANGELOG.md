# Changelog

All notable changes to BlackFlag HD will be documented in this file.

## [1.0.0] - 2025-01-25

### Added
- Initial release for heavy-duty vehicles (Class 4-8)
- J1939 protocol implementation (SAE J1939-71/73)
- J1708/J1587 legacy protocol support
- 150+ heavy-duty vehicle database
- 25 manufacturer support (Freightliner, Peterbilt, Kenworth, Volvo, Mack, etc.)
- 9-pin and 6-pin Deutsch connector documentation
- DPF/DEF/EGR diagnostic capabilities
- Aftertreatment system analysis tools
- Engine brake calibration support
- Transmission shift map editing
- PTO configuration tools
- Fleet management integration

### Features by Category

#### Engine Systems
- Cummins X15/ISX15 full support
- Detroit DD13/DD15/DD16 calibrations
- PACCAR MX-11/MX-13 diagnostics
- Volvo D11/D13 engine support
- Navistar A26/N13 compatibility

#### Transmission
- Eaton Fuller automated manual
- Allison automatic calibrations
- Detroit DT12 support
- Volvo I-Shift programming

#### Emissions
- DPF forced regeneration
- DEF quality monitoring
- EGR position control
- SCR efficiency tracking
- NOx sensor diagnostics

#### Heavy Equipment
- Caterpillar C13/C15 support
- John Deere PowerTech engines
- Case IH/New Holland agriculture
- Construction equipment modules

### Documentation
- J1939 PGN reference guide
- SPN/FMI code database
- Connector pinout diagrams
- Protocol timing specifications
- Wiring harness schematics
